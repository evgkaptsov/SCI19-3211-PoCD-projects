This repository contains programs and projects related to the course SCI19 3211 *Principles of Compiler Design*, taught by Dr. Evgenii Kaptsov at Suranaree University of Technology, starting from the third semester of the 2025/26 academic year (Thai: 2568/2569).

Please note that the code is intended for educational purposes. It is deliberately simplified to improve readability and facilitate student understanding, and is therefore recommended for educational use only.
